# 🏦 Bank Management System (Java)

## 📌 Description
This is a simple console-based Bank Management System built using Java.

## 🚀 Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Exit System

## 🛠 Technologies Used
- Java
- Scanner Class
- OOP Concepts

## 🎯 Learning Outcome
This project helps in understanding:
- Classes and Objects
- Methods
- Switch Case
- Loops
- Basic Banking Logic

## ▶️ How to Run
1. Open the file in any Java IDE or VS Code
2. Compile the program
3. Run the program
4. Follow the menu options
